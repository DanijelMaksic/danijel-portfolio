# useBook App

App which uses REST API to fetch book data. Users can browse and save the books.

<HeroImage image="use-book" title="useBook App" />

UseBook is a browsing app for finding and saving books. It is built with React and Tailwind CSS. Google books API is used for data fetching.

<br />
<br />

## Searching for a book

<br />

<ArticleImage
   image={useBookImg2}
   alt="Searching for a book"
   isResponsive={false}
/>

<br />

When user searches for a book, a list of books will appear on the left, from which each book can be selected. On the right side, details of the currently selected book will be displayed, from where user can save the book.
