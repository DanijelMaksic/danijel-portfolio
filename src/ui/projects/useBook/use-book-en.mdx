# useBook App

### App that uses REST API to fetch book data. Users can browse and save the books.

<HeroImage image={UseBookImg} title="useBook App" />

UseBook is a browsing app for finding and saving books. It is built with React and Tailwind CSS. Google Books API is used for data fetching.
