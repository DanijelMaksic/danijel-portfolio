# useBook Апликација

### Апликација која користи REST API за преузимање података. Корисници могу да разгледају и сачувају књиге.

<HeroImage image={UseBookImg} title="useBook App" />

UseBook је апликација где је могуће тражити и сачувати књиге. Направљена је са React-ом и Tailwind CSS-ом. Google books API је коришћен за преузимање података о књигама.

Када корисник напише име књиге, на левој страни екрана ће се појавити листа са које је могуће одабрати књигу. На десној страни ће бити приказани детаљи о одабраној књизи, одакле корисник може сачувати књигу.
