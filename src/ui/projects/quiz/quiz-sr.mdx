# Историја Старог Света - Квиз

Квиз направљен да тестира твоје знање историје старог света.

<HeroImage image={QuizImg} title="Историја Старог Света - Квиз" />

Историја Старог Света - Квиз је апликација коју покреће React и која омогућава корисницима да тестирају своје знање и да се такмиче за највиши резултат. Питања и највећи резултат су сачувани у JSON фајловима и преузети су коришћењем mock API-а који пружа JSON-Server.

<br />

## Подешавање квиза

Почетна страница приказује пар информација о временском периоду да би припремила корисника за оно што долази. Ту су четири нивоа тежине, са различитим бројем питања за сваки од њих.

## Квиз страница

Сваки ниво тежине награђује другачију количину поена, дакле највећи ниво тежине даје највише поена. Корисник такође мора да пази на тајмер (колико квиз траје зависи од броја изабраних питања).
