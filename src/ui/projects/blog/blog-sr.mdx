# Блог апликација

### Једноставна блогинг апликација са функцијом обележавања артикала. Направљена са ванила JavaScript-ом користећи MVC архитектуру.

<HeroImage image={BlogImg} title="Блог Апликација" />

Мој Блог је ванила JavaScript апликација која омогућава својим корисницима да пишу и објављују артикле. Направљена је у HTML-у, са ванила CSS-ом и JavaScript-ом. Апликација је структуисана користећи MVC (Model-View-Controller) архитектуру, која раздваја апликацију у три дела зарад лакшег разумевања и мање шпагети кода.
