# Blog App

A simple blogging app with a bookmark feature. Built with vanilla JavaScript using MVC architecture.

<HeroImage image={BlogImg} title="Blog App" />

My Blog is a vanilla JavaScript app which enables the user to write articles. It is built in HTML, with vanilla CSS and JavaScript. App is structured in MVC (Mode-View-Controller) architecture model, which splits the application into three parts for easier understanding and less spaghetti code.

When add article button is clicked, a modal will pop out. In this modal window user can write the article, along with deciding on a tag.
