# Judgment Вебсајт Редизајн

### Комплетан редизајн промоционалног вебсајта за награђивану видео-игру Judgment.

<HeroImage image={JudgmentImg} title="Judgment Вебсајт Редизајн" />

Judgment вебсајт је упеглан редизајн оригиналног вебсајта који представља хит детективску драму у власништву компаније SEGA. Направљено са **HTML**-ом, ванила **CSS**-ом, и са **JavaScript**-ом за неке функционалности.

<br />

Вебсајт је стилизован тако да што више одговара модерној ноар атмосфери игре, са пар CSS трикова да истакну дизајн. Прилагодљиви дизајн је искоришћен да би омогућио прегледност на многим уређајима.
