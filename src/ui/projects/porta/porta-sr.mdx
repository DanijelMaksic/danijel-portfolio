# Порта Вебстајт

### Вебсајт брошура за фикционалну столарску радњу, која представља њену понуду.

<HeroImage image={PortaImg} title="Порта Вебсајт" />

Порта је презентациони вебсајт који приказује шта ова столарска радња има да понуди. Вебсајт је написан у HTML-у заједно са CSS екстензијом SASS, са мало JavaScript-а за одређене функционалности.

<br />

Порта вебсајт спаја модерни и занатски изглед, са прилагодљивим дизајном који омогућава корисницима да прегледају сајт на мноштву уређаја.
