# Ochrid Gallery E-commerce

A Next.js e-commerce website for a fictional gallery. Includes checkout functionality and user authentication.

<HeroImage image={OchridGalleryStoreImg} title="Ochrid Gallery E-commerce" />

Ochrid Gallery is a webstore that enables its users to browse and order icons. Website is powered by Next.js which provides fast initial load and server-side rendering. It is styled with Tailwind CSS and can be used on variety of devices thanks to the responsive design. App's data is stored in Supabase database. Deployed on Vercel.

## Checkout process

After one or more icons have been added to cart, user can visit the shopping cart page. This page contains the cart management, summary of the order and some featured trending products. Here user can start the checkout process, but in order to proceed to the next part, user needs to register/log in using his Google profile.
